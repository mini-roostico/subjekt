ResolvedSuite

data class ResolvedSuite(val originalSuite: Suite, val resolvedSubjects: Set<ResolvedSubject>)

Represents a resolved Suite. A ResolvedSuite contains the whole result of a Subjekt elaboration, working as a container of the main result objects, the ResolvedSubjects.
